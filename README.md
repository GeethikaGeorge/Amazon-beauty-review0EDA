# Amazon-beauty-review0EDA
Exploratory Data Analysis and Cleaning on Amazon Beauty Reviews Dataset
This project performs a complete data cleaning, pre-processing, and exploratory data analysis (EDA) of the Amazon Beauty Reviews Dataset consisting of more than 701,000 customer reviews from Amazon’s beauty and personal care category.

The goal is to uncover meaningful insights about customer behaviour, product ratings, review sentiment, yearly trends, and helpfulness patterns.
Dataset Information

Source: Hugging Face
https://huggingface.co/datasets/jhan21/amazon-beauty-reviews-dataset

Rows: 701,528

Columns: rating, title, text, images, asin, parent_asin, user_id, timestamp, helpful_vote, verified_purchase
